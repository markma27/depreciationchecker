Depreciation Checker Extension Privacy Policy

Data Collection and Storage

  • This extension collects asset details and depreciation calculations entered by users
  • All data is stored locally in your browser using Chrome's storage API
  • No data is transmitted to external servers
  • No personal information is collected

Data Usage

  • Collected data is used solely for providing depreciation calculation functionality
  • Asset history and calculations are maintained locally for user reference
  • Data is automatically cleared when you clear your browser data

Third-Party Access

  • No data is shared with third parties
  • All information remains private to your browser
  • No analytics or tracking services are used

External Resources

  • The extension loads fonts from Google Fonts for user interface styling
  • Icons are loaded from Font Awesome CDN for user interface elements
  • These resources are used solely for visual presentation and do not access any user data
